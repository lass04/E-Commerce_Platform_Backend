This is Node.js Backend for an E-Commerce Platform 

Technologies :
* Node.js
* Express
* MongoDB Atlas 

Features :
* RESTful APIs
* JWT Authentication 
* Role-Based Authorization
* bcrypt Password Hashing
* AI Integration
* Rate Limiting
* Cors Configuration
* Helmet Configuration (To prevent XSS,ClickJacking...) 
* Products Filtering & Pagination
